# GoBig

Foundational framework for building apps in the organization.

- Tech stack: Astro, Better Auth, Drizzle ORM, Stripe, Chakra UI, Tailwind CSS, TypeScript
- Dev server: `http://localhost:4321`

## Getting Started

```bash
# From repo root (recommended)
# Install app deps
bun install

# Run the app
bun run dev

# In a new terminal, install docs deps
(cd docs && bun install)

# Run the docs site at http://localhost:3000/gobig/
(cd docs && bun run dev)
```
